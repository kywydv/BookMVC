Постановка задачі

Метою лабораторної роботи є розробка вебдодатку інтернет-магазин книжок



Застосунок реалізує функціонал онлайн-книгарні, а саме:

\- перегляд каталогу книг, пошук, фільтрацію за автором/жанром;

\- перегляд детальної інформації про книгу;

\- додавання книг у кошик, зміну кількості, очищення кошика;

\- оформлення замовлення (вказання адреси доставки, способу оплати);

\- реєстрацію та автентифікацію користувача, перегляд профілю та історії

&#x20; замовлень.



Структура предметної області

Основні сутності (детальніше — у діаграмі бази даних):



| Сутність     | Призначення                                   |

|--------------|------------------------------------------------|

| Користувач   | Дані облікового запису покупця                |

| Книга        | Картка товару (назва, автор, жанр, ціна тощо) |

| Автор        | Автор книги                                   |

| Жанр         | Жанр книги                                    |

| Видавництво  | Видавництво книги                             |

| Кошик        | Позиції книг, додані користувачем в кошик     |

| Замовлення   | Оформлене замовлення на основі кошика         |
